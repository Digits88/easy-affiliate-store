The Easy Affiliate Store Plugin for Wordpress lets you quickly add an affiliate store to your site.

== Description ==

The Easy Affiliate Store Plugin for Wordpress lets you quickly add an affiliate store to your blog.

It aims to get you up and running as soon as possible, by giving you a store which requires minimal setup and is ready to go almost as soon as it is installed. The store has a simple design which integrates with your blog's own theme.

Some features include:

* Store front shows featured products and latest products by "department" category
* Department pages show all products of one type
* Create as many departments as you like
* Product page showcases a product in a natural blog post format
* Adds prominent call-to-action button with affiliate link
* Automatically adds Amazon affiliate to Amazon.com links
* Call-to-action button shows favicon of target site
* Products can be configured to show up on the homepage, categories and search. Or not.
* Optional custom sidebar on products page

For full details, please see the [Easy Affiliate Store webpage](https://www.danonwordpress.com/easy-affiliate-store-plugin-for-wordpress/).

== Installation ==

To install the store plugin on your site, do the following:

1. In your WordPress Dashboard go to "Plugins > Add New"
2. Search for "Easy Affiliate Store"
3. Click "Install Now"
4. Click "Activate Plugin"

That's it, the Easy Affiliate Store should now be installed on your blog.

== Frequently Asked Questions ==

= I use Amazon Associates. Can I automatically add my affiliate to Amazon.com links? =

Yes, there is an option in the settings page ("Settings > Easy Affiliate Store" in your Wordpress admin panel) to add your affiliate ID. This will automatically be added to the product URL when you link to Amazon.com products.
